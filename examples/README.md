# AgentBase Examples

This directory contains small, focused examples demonstrating how to use the AgentBase platform.  
Each subfolder groups examples by purpose, complexity, or execution model.

The examples are intentionally minimal so they are easy to read, extend, and validate against the platform schemas.

---

## Workflow Examples (v1)

**Path:** `/examples/workflows/`  
These examples illustrate the original workflow model using Execution Semantics v1.  
They are ideal for understanding the basics of tasks, sequencing, and state mapping.

Use these if you want a simple, linear introduction to workflows.

---

## Workflow Examples (v2)

**Path:** `/examples/workflows-v2/`  
These examples demonstrate the advanced capabilities introduced in Workflow Schema v2 and Execution Semantics v2, including:

- parallel execution  
- conditional guards  
- race semantics  
- fallback routing  
- explicit dependency graphs  

Use these when building more expressive or orchestration‑heavy workflows.

---

## Task Examples

**Path:** `/examples/tasks/`  
Examples showing how to define individual tasks, including inputs, outputs, and metadata.  
Useful when creating new capabilities or testing task-level behavior.

---

## Error Examples

**Path:** `/examples/errors/`  
Examples illustrating validation errors, malformed requests, and error-handling patterns.  
Helpful for debugging and for understanding how AgentBase enforces schema correctness.

---

## Agent Request Examples

**Path:** `/examples/agent-requests/`  
Examples of how agents communicate with AgentBase using request envelopes.  
These are useful for building or testing agent integrations.
